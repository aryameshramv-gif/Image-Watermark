# Image Watermarker (Tkinter,PIL)
---

This program takes an input image and watermark and overlays it on top of eachother
It utilises a UI canvas to display the result image and provides a download button 

---
## Features 
- Takes two input images (one main and one watermark)
- creates the input images into thumnails of different sizes.
- overlays both thumbnails on top of eachother.
- displays the result on a UI 
- download button to save the image into a specified folder.
